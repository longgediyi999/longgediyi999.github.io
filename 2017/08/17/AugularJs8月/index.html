AugularJs8月

发表于 2017-08-17 | 分类于 AngularJs 
声明应用:位html标签添加ng-app属性  可以指定一个应用 表明此标签包裹的内容都属于应用的一部分

声明模块: angular.module('myApp',[]);两个参数: 模块名称 依赖列表

控制器: controller 控制器是连接模型和视图的桥梁 所以当我们定义好了控制器以后就定义好了模型和视图
controller关联到视图是上,通过为html标签添加ng-controller属性并赋值控制器名称,确立关联关系

大致步骤:

body中: 定义应用名称ng-app="a123"
        定义控制器名称:ng-controller="a456";
        上述子元素中{{ counter(元素变化) }}
script中: var app = angular.module("a123",[]);
          app.controller('a456',function($scope){
                $scope.counter = "1";  //控制变量
          })


传参大致步骤:
<body>
<div ng-app="app" ng-controller="FirstController">
    <h4>asdfaasdf</h4>
    <button ng-click="add(1)" class="button">Add</button>
    <button ng-click=subtract(1)>Subtract</button>
    <h4>Current count: {{ counter }}</h4>
</div>
</body>
